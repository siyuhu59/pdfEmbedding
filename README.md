# pdfEmbedding
주의 사항: 
`아직 프로젝트에 처리되지 않은 오류가 많습니다`


# 설치 방법

1. Git 프로젝트 클론

```bash
git clone https://github.com/siyuhu59/pdfEmbedding.git
cd pdfEmbedding
```

2. Python library 설치 및 실행

```
pip install -r requirements.txt
python main.py
```
